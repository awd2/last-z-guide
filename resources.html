Farm Accounts (Late Game)

When resources become scarce in late game, create farm accounts:

What is a Farm Account?

  • Secondary account focused only on gathering
  • Transfers resources to your main account
  • Essential for F2P players in late game

How to Set Up

  1. Create new account on same server
  2. Join same alliance as main
  3. Focus on gathering buildings and march capacity
  4. Send gathered resources to main via alliance transfers

Tips for Farm Accounts

  • Keep HQ low â€” no need to upgrade past gathering unlocks
  • Max gathering research
  • Build multiple farms if needed (1-2 is enough)
  • Don't invest in combat â€” only gathering

Skip Resource Buildings

Common mistake: Building and upgrading resource production buildings.

Why Resource Buildings Are Bad

  • Terrible output â€” produce very little per hour
  • Gathering is 10-50x better â€” one gathering trip = days of production
  • Waste builder time â€” builders should focus on HQ prerequisites
  • Waste resources â€” upgrading them costs more than they produce

What Buildings to Skip

  • Farmhouse (food production)
  • Residence (wood production)
  • Wind Turbine (electricity production)
  • Smelting Plant (steel production)

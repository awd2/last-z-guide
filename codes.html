Can I Redeem Last Z Codes on iPhone or Android?

Yes. The redemption flow is the same on iPhone and Android because codes are claimed through the official Gift Center website, not inside the game app.

  • iPhone: open the Gift Center in Safari or another browser, then paste your UID and code.
  • Android: open the Gift Center in Chrome or another browser, then paste your UID and code.
  • In-game: not supported for code redemption.

How to Find Your UID in Last Z

Your UID is required for every code redemption. If the UID is wrong, the code will fail even if the code itself is still active.

Before redeeming a code, you need your UID (User ID):

  1. Open Last Z: Survival Shooter
  2. Tap your Avatar (top-left corner)
  3. Go to Settings
  4. Tap Copy ID
  5. Your UID is now copied to clipboard

How to Redeem Last Z Codes Step by Step

The safest way to redeem a code is to copy your UID directly from the game and paste the code exactly as shown. Most failed redemptions happen because of typos, wrong letter case, or an incorrect UID.

  1. Open https://last-z.com/giftCenter/#/login
  2. Paste your UID into the UID field
  3. Enter the Gift Code exactly as shown
  4. Tap Confirm
  5. Open the game
  6. Check your in-game mailbox for rewards

Why Your Last Z Code Is Not Working

Most failed redemptions come from one of these issues. Check them in this order before assuming the code is dead.

  1. You are not using the official Gift Center. Codes are not redeemed inside the game.
  2. Your UID is wrong. Copy it directly from Settings instead of typing it manually.
  3. The code was entered incorrectly. Use copy and paste to avoid letter-case mistakes.
  4. The code was already used on this account. Most codes are one-time use per account.
  5. The code expired. Event and milestone codes often disappear quickly.

Where Are My Last Z Code Rewards?

Rewards are delivered to your in-game mailbox, not shown instantly on the Gift Center page. After redeeming, open the game and check your mail. If nothing appears immediately, wait a few minutes and restart the game once.

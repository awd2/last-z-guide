How to Find Your UID in Last Z

Your UID is required for every code redemption. If the UID is wrong, the code will fail even if the code itself is still active.

Before redeeming a code, you need your UID (User ID):

  1. Open Last Z: Survival Shooter
  2. Tap your Avatar (top-left corner)
  3. Go to Settings
  4. Tap Copy ID
  5. Your UID is now copied to clipboard

How to Redeem Last Z Codes Step by Step

The safest way to redeem a code is to copy your UID directly from the game and paste the code exactly as shown. Most failed redemptions happen because of typos, wrong letter case, or an incorrect UID.

  1. Open https://last-z.com/giftCenter/#/login
  2. Paste your UID into the UID field
  3. Enter the Gift Code exactly as shown
  4. Tap Confirm
  5. Open the game
  6. Check your in-game mailbox for rewards

Why Your Last Z Code Is Not Working

Most code failures come from one of four causes: the code expired, the UID is wrong, the code was already used on that account, or the text was entered with a typo.

  • Code has expired
  • Code already redeemed on your account
  • Wrong UID entered
  • Typo or incorrect letter case
  • Code not officially released yet

FAQ

Where do I redeem Last Z codes?

On the official Gift Center website: last-z.com/giftCenter. Codes cannot be redeemed inside the game.

What is the official Last Z Gift Center login page?

Use the official Gift Center login page: last-z.com/giftCenter/#/login.

How do I find my UID in Last Z?

In the game: Avatar (top-left) → Settings → Copy ID. Your UID will be copied to your clipboard.

Why is my Last Z code not working?

Common reasons: the code expired, you already redeemed it, the UID is wrong, or there is a typo. Double-check your UID and copy-paste the code exactly.

Can I redeem Last Z codes inside the game?

No. Last Z codes are redeemed on the official Gift Center website, not inside the game client.

How do you verify Last Z codes?

We only mark codes Active after they appear in the official Last Z Discord announcements and are confirmed via the Gift Center.

Where are my rewards?

Rewards are sent to your in-game mailbox. Open the game and check your mail. If you don't see them, restart the game and wait a few minutes.

How often are new codes released?

New codes come out during game updates, special events, holidays, and milestone celebrations. Follow official social media for announcements.

Can I use codes on multiple accounts?

Yes, each code can be used once per account. If you have multiple accounts, you can redeem the same code on each one using each account's unique UID.
